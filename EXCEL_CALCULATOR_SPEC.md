# 鸣潮排轴DPS计算器 - 满血Excel功能规格书

## 一、项目概述

实现一个专业级的网页版Excel DPS计算器，支持完整的Excel功能，包括公式计算、单元格引用、函数等。

## 二、技术方案选型

### 方案对比：

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **Univer** (Luckysheet升级版) | 功能完整、持续维护、支持协同 | 学习成本较高 | ⭐⭐⭐⭐⭐ |
| **FortuneSheet** | TypeScript、现代化、React/Vue支持 | 社区较小 | ⭐⭐⭐⭐ |
| **纯自研** | 完全定制、灵活 | 开发周期长、维护成本高 | ⭐⭐⭐ |

### 最终选择：**Univer + 自定义DPS计算模块**

理由：
1. Univer是Luckysheet的官方升级版，持续维护
2. 支持完整的Excel公式和函数
3. 支持单元格引用、数据验证、条件格式
4. 支持导入导出Excel文件
5. 可以通过API与自定义计算逻辑集成

### 步骤3：实现核心计算公式

#### 3.1 角色面板计算公式
```
面板暴击 = SUM(基础暴击, 武器暴击, 套装暴击, 主副词条暴击, 共鸣链暴击, 队友暴击)
面板暴击 = MIN(面板暴击, 100%)

面板爆伤 = SUM(基础爆伤, 武器爆伤, 套装爆伤, 主副词条爆伤, 共鸣链爆伤, 队友爆伤)

双爆区 = 1 + 面板暴击 * (面板爆伤 - 1)
```

#### 3.2 伤害计算公式
```
伤害 = 次数 * 白值 * 倍率 
     * (1 + 攻击区) 
     * (1 + 加成区) 
     * 双爆区
     * (1 + 加深区)
     * 防御区
     * 抗性区
     * (1 + 倍率提升)
     * (1 + 易伤区)
     * (1 + 最终伤害提升)
```

#### 3.3 防御区计算公式
```
角色防御 = 800 + 8 * 角色等级
怪物防御 = (792 + 8 * 怪物等级) * (1 - 减防%) * (1 - 无视防御%)
防御区 = 角色防御 / (角色防御 + 怪物防御)
```

#### 3.4 抗性区计算公式
```
IF 抗性 < 0:
    抗性区 = 1 + |抗性| / 2
ELSE IF 抗性 <= 80%:
    抗性区 = 1 - 抗性
ELSE:
    抗性区 = 1 / (5 * (抗性 + 0.2))
```

## 五、效应伤害和震谐伤害详细说明

### 效应伤害类型（5种）
1. **光躁**: 光属性持续伤害
2. **聚爆**: 火属性爆发伤害
3. **霜渐**: 冰属性持续伤害
4. **电磁**: 雷属性持续伤害
5. **风蚀**: 风属性持续伤害

每种效应都有：次数、白值、倍率、攻击区、双爆区、加深区、防御区、抗性区、倍率提升、易伤区、最终伤害提升

### 震谐伤害类型（4种）
1. **角色1震谐**: 角色1的震谐技能伤害
2. **角色2震谐**: 角色2的震谐技能伤害
3. **角色3震谐**: 角色3的震谐技能伤害
4. **谐度击破**: 特殊震谐伤害

每种震谐都有相同的计算字段
