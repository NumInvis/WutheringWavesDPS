# 鸣潮排轴DPS计算器 - 技术方案文档 V2.0

## 1. 项目概述

### 1.1 项目目标
将分散在五个独立界面的功能模块整合至单一操作界面，每个模块以可交互的方框组件形式呈现，具备完整的Excel级计算功能。

### 1.2 核心功能模块
- **角色面板模块**：攻击区、加成区、暴击区、爆伤区、加深区、副词条
- **伤害计算模块**：动作伤害计算、各乘区显示
- **效应伤害模块**：光躁、聚爆、霜渐、电磁、风蚀
- **震谐伤害模块**：角色1震谐、角色2震谐、角色3震谐、谐度击破
- **环境参数模块**：怪物等级、抗性、减防等
- **汇总统计模块**：各角色DPS、总DPS、总伤害

---

## 2. 技术方案选择

### 2.1 推荐方案：Path A + Path B 混合方案

#### 方案说明
- **前端**：使用 **Luckysheet** 或 **Univer** 作为核心表格组件
- **后端**：保持现有 Flask + SQLite 架构
- **集成方式**：单界面模块化布局，每个模块是一个独立的表格或组件

#### 技术选型理由
1. **Luckysheet/Univer 是成熟的开源Excel前端库**
   - 完全免费且开源
   - 具备完整Excel核心功能
   - 支持公式计算、单元格引用、数据格式化
   - 支持.xlsx导入/导出
   - 活跃的社区维护

2. **单界面模块化设计**
   - 使用 CSS Grid/Flexbox 布局
   - 每个功能模块是一个可独立拖拽/调整大小的面板
   - 面板间数据联动

### 4.4 里程碑 4.4：伤害计算模块 (3-4天)

#### 功能要求
- 动作数据输入（动作名、次数、倍率）
- 各乘区实时计算显示
- 伤害自动计算
- 支持添加/删除动作

#### 计算逻辑
```javascript
// 注册自定义函数
luckysheet.registerFunction({
  name: 'WUWA_ATK_ZONE',
  func: (charId) => {
    return getCharData(charId).atkBonus;
  },
  description: '获取角色攻击区'
});

luckysheet.registerFunction({
  name: 'WUWA_DEF_ZONE',
  func: (charLevel, monsterLevel, defReduction, defIgnore) => {
    const charDef = 800 + 8 * charLevel;
    const monsterDef = (792 + 8 * monsterLevel) * 
                      (1 - defReduction / 100) * 
                      (1 - defIgnore / 100);
    return charDef / (charDef + monsterDef);
  },
  description: '计算防御区'
});
```
