## ✨ 功能特性

- 📊 **在线表格编辑** - 基于 Luckysheet，Excel 完美兼容
- ✅ **完全兼容** - 保留数据、公式、格式不变
- 👥 **社区分享** - 上传并分享你的拉表
- ❤️ **完全公益** - 免费使用，永久开源
- ⭐ **点星功能** - 为喜欢的拉表点赞
- 🔍 **搜索功能** - 按作者/编号搜索表格
- 👤 **用户系统** - 注册、登录、修改密码
- 🛠️ **管理员功能** - 置顶、删除、日志监控
- 💾 **状态保持** - 工作区状态自动保存

## 🚀 快速开始

### 后端

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 0.0.0.0 --port 14876 --reload
```

### 前端

```bash
cd frontend
npm install
npm run dev
```

## 🛠️ 技术栈

- **后端**：FastAPI + SQLite + SQLModel
- **前端**：Vue 3 + TypeScript + Element Plus + Vite
- **表格组件**：Luckysheet
